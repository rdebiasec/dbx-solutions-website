# Perguntas por seção — Aviso de Tratamento de Dados

## Escopo

**P: Para que serve o Aviso de Tratamento de Dados?**  
**R:** Para explicar como tratamos dados de clientes e dados de negócio em **projetos de clientes** — implementações, integrações e operação contínua. A privacidade de visitantes do site está na Política de Privacidade.

**P: Isso é a mesma coisa que a Política de Privacidade?**  
**R:** É relacionado, mas diferente. Política de Privacidade = visitantes do site. Aviso de Tratamento de Dados = como desenhamos práticas de dados nos projetos que construímos para clientes.

---

## Nossa abordagem

**P: Como vocês tratam dados no trabalho com clientes?**  
**R:** Usamos conteúdo de negócio aprovado como base para respostas, limitamos o acesso ao necessário em cada fluxo, desenhamos integrações para enviar contexto útil ao seu time — sem exposição aberta, documentamos o que é processado em cada projeto e alinhamos com suas políticas e regras do setor quando aplicável.

**P: Vocês usam dados aleatórios da internet para responder clientes?**  
**R:** Não — respostas para clientes vêm de informações de negócio aprovadas e controladas por você.

**P: Quem pode acessar dados em um projeto?**  
**R:** Apenas o necessário para cada fluxo de cliente — desenhamos tudo com acesso controlado.

---

## Para prospects

**P: Se contratarmos vocês, vão processar dados dos nossos clientes?**  
**R:** O escopo é definido por projeto. Documentamos o que será processado, armazenado e retido em cada contratação.

**P: Vocês conseguem alinhar com nossa política de privacidade?**  
**R:** Sim — buscamos alinhar com sua política de privacidade, termos de SMS e requisitos do setor quando aplicável.
